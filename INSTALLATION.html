Admin Panel Installation Guide

Short setup guide for buyers. This project is frontend-only (Next.js).

1) Requirements

  • Node.js (latest LTS recommended)
  • npm (included with Node.js)

2) Install Dependencies

npm install

3) Create Environment File

Copy .env.example to .env.local.

Windows PowerShell:

Copy-Item .env.example .env.local

macOS/Linux:

cp .env.example .env.local

Set values like this:

NEXT_PUBLIC_APP_NAME=Admin Panel
NEXT_PUBLIC_API_BASE_URL=http://localhost:4000/api

4) Run the Project

npm run dev

Open http://localhost:3000 in your browser.

5) Production Commands

npm run build
npm run start

Available Scripts

  • npm run dev - Development server
  • npm run build - Production build
  • npm run start - Start production server
  • npm run lint - Lint checks
  • npm run test - Run tests
  • npm run test:watch - Run tests in watch mode
  • npm run format - Check formatting
  • npm run format:write - Auto-format code

Important Note

This package does not include backend/database code. Connect NEXT_PUBLIC_API_BASE_URL to your own API.
